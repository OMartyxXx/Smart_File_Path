SmartFilePath est un addon Blender conçu pour simplifier et sécuriser la gestion des chemins de rendu. Il permet de configurer en quelques clics les chemins de sortie pour le rendu final (PNG, EXR, Cryptomatte) ainsi que pour les previews (MP4 via viewport render), en générant automatiquement des noms de fichiers structurés incluant le nom du projet, la caméra active et le numéro de version. L'addon intègre également des outils de création de rig caméra, de gestion du frame range et du mist pass, ainsi qu'une compatibilité avec Deadline pour l'envoi de jobs de rendu en farm.
